Sweetshop - дипломный проект по теме: "Мобильное приложение для клиентов кондитерского магазина"

Для правильного запуска проекта необходимо:

1. Иметь установленные:
 Android Studio (желательно версии FOX от 2020) + Android SDK 30
 Microsoft Visual Studio 2019 + расширение Conveyor by Keyoti ( любые версии от 2021)
 Microsoft SQL Server Management Studio v18

2. Используемые проекты:
CakeClient2 - проект на Anroid Studio
CakeClientApi - проект web-api в Microsoft Visual Studio
DBase - скрипт базы данных

После скачивания всех файлов, в Api проекте необходимо проверить строку подключения с БД и запустив Api убедиться в работе расширения.
Затем, если необходимо, изменить IP адрес в проекте Anroid Studio.
